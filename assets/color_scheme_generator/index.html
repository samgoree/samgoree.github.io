Color Generator

This tool uses a fancy probabalistic model to choose colors, then generates a triangle pattern using those colors! There are three options for model:

  • ColourLovers: Based on data from ColourLovers.com. This one usually looks the best!
  • Templates: Based on common color templates. This one sometimes looks wierd.
  • ColorMind: Based on data from colormind.io. This one is boring sometimes.

There's also a "Wierdness" parameter. This controls how much the colors are allowed to vary.
